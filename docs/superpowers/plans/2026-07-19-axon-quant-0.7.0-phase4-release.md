## Phase 4 范围

> 注:0.7.0 0.6.0 plan 提过 Phase 4 但仅 minimal(只 `max_leg_pair_net_exposure` 一个字段),且跳到 0.8.0。

### 4.1 `PortfolioRiskEngine::delta_exposure()` per instrument

- 新增 `axon_risk::portfolio::PortfolioRiskEngine` 结构
- `delta_exposure(&portfolio) -> HashMap<Instrument, f64>` — 每个 instrument 的 delta 暴露
- **delta 定义**:
  - Spot instrument: `delta = position_qty`(1 unit = 1 unit underlying)
  - Swap/Perp instrument: `delta = position_qty`(线性合约,1 contract = 1 unit notional)
- `portfolio_delta = Σ delta[instrument]`

### 4.2 `PortfolioRiskEngine::gamma_exposure()` per instrument + 总 gamma

- `gamma_exposure(&portfolio) -> HashMap<Instrument, f64>` — 每个 instrument 的 gamma 暴露
- **gamma 定义**:
  - Spot instrument: `gamma = 0`(线性)
  - Swap/Perp instrument: `gamma = 0`(在无 IV / 无 mark 历史场景下无法精确算)
- 跨 instrument gamma 协方差:本 plan **不实现**(需要 mark 历史窗口,BacktestEngine 暂未持久化 mark 历史),留 TODO 0.8.0
- `total_gamma = Σ gamma[instrument]`

### 4.3 `RunResult.risk_metrics: dict` 暴露

- 新增 `RunResult.risk_metrics: RiskMetricsReport` 字段
- `RiskMetricsReport` 结构:
  ```rust
  pub struct RiskMetricsReport {
      /// per-leg delta:`instrument -> delta`
      pub per_leg_delta: HashMap<Instrument, f64>,
      /// 组合总 delta:`Σ per_leg_delta.values()`
      pub portfolio_delta: f64,
      /// per-leg gamma:`instrument -> gamma`
      pub per_leg_gamma: HashMap<Instrument, f64>,
      /// 组合总 gamma
      pub total_gamma: f64,
      /// vega(暂时 0.0,无 IV 源)
      pub vega: f64,
      /// 多 leg Sharpe(沿用 `RunResult.sharpe_ratio` 的多 leg 实现)
      pub sharpe_with_legs: f64,
  }
  ```
- `RunResult` `Default` 改:空 `HashMap` + 0 值
- PyO3 binding 暴露 `risk_metrics: dict`

### 4.4 单测 + e2e

- 单元测试(`crates/axon-risk/src/portfolio.rs`):
  - 空 portfolio:`portfolio_delta == 0`,`total_gamma == 0`
  - 单 leg +1 BTC spot:delta = +1
  - 跨 spot + perp delta-neutral:portfolio_delta ≈ 0
  - 多 leg + 1 BTC / -1 BTC:portfolio_delta == 0
- e2e 测试(`crates/axon-backtest/tests/e2e_risk_metrics.rs`):
  - spot + perp delta-neutral 跑回测
  - `RunResult.risk_metrics.portfolio_delta ≈ 0`
  - 验证 `risk_metrics.per_leg_delta[instrument]` 字段精确
  - 验证 `RunResult.risk_metrics` 暴露到 Python 端
