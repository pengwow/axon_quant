## 编写规范

每条 ADR 包含以下部分：

1. **状态**：提案中 / 已接受 / 已废弃 / 已取代
2. **背景**：驱动该决策的问题
3. **决策**：选择的具体方案
4. **后果**：正面与负面后果、风险
5. **参考**：相关文档或外部链接

新增 ADR 时使用下一个序号（`NNNN-简短描述.md`）并在索引表中追加。
